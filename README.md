# Linux wallpaper

## 介绍

使用electron-forge构建的桌面动态壁纸

支持图片，视频以及网页

## 快速开始

```bash
# 安装依赖
yarn install

## 启动

yarn start

## 打包

yarn make

```

### deb&rpm

```json
 {
          "name": "@electron-forge/maker-deb",
          "config": {}
        },
        {
          "name": "@electron-forge/maker-rpm",
          "config": {}
        }
```
